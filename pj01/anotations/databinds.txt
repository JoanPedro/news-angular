Databinding = Communication

Business Logic ==> Template HTML

          Output Data ==>
  One-way from data source to view target

    1. String interpolation -> {{ data }};
    2. Property binding -> [property] = "data";

      <== React to (User) Events
  One-way from view target to data source

  1. Event Binding -> (event) = "expression";

          Combination of Output and Event binding
                        <==>

  1. Two-way Databinding -> [(ngModel)]="data"

String interpolation: Mostrar algo em Tela {{ data }}
Property binding: Mudar alguma propriedade de um elemento HTML / diretiva de um componente [ property ]
Event binding: Interceptar eventos ( eventName )
